Summary Latitude vs x

The purpose of this project was to analyse how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After obtaining the data from the api it was then appended to our table of randomly selected cities. Factors examined include humidity, cloudiness, wind speed and maximum temperature Matplotlib was used to plot the various aspects of the weather vs. the city latitude.

This site provides the source data and visulisations with a short description determinating whether there is any correlation.
